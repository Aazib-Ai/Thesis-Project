                     1. PROJECT OVERVIEW
================================================================================

This project is a FULLY FUNCTIONAL implementation of an Optimized Hybrid 
Encryption System that combines two powerful cryptographic techniques:

  â€¢ AES-256-GCM: Military-grade symmetric encryption for fast protection 
                  of textual/PII data (names, addresses, IDs)
                  
  â€¢ CKKS Homomorphic Encryption: Advanced cryptographic scheme that allows 
                                  mathematical computations DIRECTLY on 
                                  encrypted data without ever decrypting it

The system demonstrates that healthcare organizations can perform meaningful 
statistical analysis on sensitive patient data while the data NEVER leaves 
its encrypted state - a revolutionary approach to data privacy.

- Encryption is ~1000x slower than AES
